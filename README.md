## ğŸ“ Overview

Food_Ease is an Online Food Ordering Web Application built using **Advanced Java (JSP + Servlets)** and **MySQL**.  
It allows users to browse food items, place orders, and track order history, while admins manage restaurants, menus, and orders.

The project demonstrates strong concepts of **MVC architecture**, **CRUD operations**, **session management**, **form validation**, and **role-based authentication**.
